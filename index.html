  • artifacts/{slug}/manifest.json
  • artifacts/{slug}/*.md (7 surfaces)

Agent analyzes artifacts. Drafts 10 experiments, competitors, revenue leaks. No target_report.md.

Eval system — automatic vs manual

Automatic After every audit

  • Structural eval — eval/eval.py via post_audit.py
  • Grounded checks — SSL, PSI, sitemap (before write)
  • Cross-audit — gingerpeople vs zenrojas
  • HTML render — template, no LLM

Manual Self-improvement loop

  • pattern_tracker.py — recurring flags across stores
  • calibrate_rubric.py — outcome-driven rubric patches
  • run_full_eval.py — full verdict JSON
  • eval/outcomes/ — human merchant experiment data

Truth hierarchy: experiment outcomes > grounded checks > judge consensus > human spot-check. See EVAL_LOOP.md for the Month 1–3 autonomy plan.
